## üöÄ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16+)
* [npm](https://www.npmjs.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Agent-8947/rgb-cmyk-animation.git
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

---

## üìñ Usage Guide

1. **Upload:** Drag and drop your **RGB** version into the left zone and **CMYK** version into the right zone.
2. **Analysis:** Use the manual slider or auto-animation to inspect color shifts.
3. **Customize:** Adjust the UI elements using the control panel to fit your presentation style.
4. **Export:** Click "Generate GIF" to download a perfectly looped comparison for social media or client approval.
